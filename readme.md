// *Small Business Webpage*

Created a landing page for a small, imaginary business using HTML and CSS for layout and design, and JavaScript for DOM manipulation and event handling. The site features a hero section, navigation menu, service cards highlighting bakery services, contact information, and an interactive call-to-action (CTA) headline that users can update using JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript
- Live Server (To see webpage in action)
- AI assistance (For advice on how to spruce up website designs)

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder in Visual Studio Code.
3. Make sure the following files are in the same folder:
   - `index.html`
   - `style.css`
   - `cc_7.js`
4. Install the **Live Server** extension if it is not already installed.
5. Right-click on `index.html` and select **Open with Live Server**.

## What I Learned

While building this project, I gained more experience creating responsive web pages using HTML and CSS. I learned how to organize content with HTML, apply hover effects and responsive design techniques, and style a webpage to match a specific theme. I also strengthened my understanding of JavaScript by using the DOM to select elements, attach event listeners, and dynamically update webpage content based on user input.
